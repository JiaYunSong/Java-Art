• 一、目的

    (1).进一步熟悉Java开发环境的使用。

    (2).通过实践练习初步了解Java程序的构成和编写方法。

    (3).常用Java类（API）使用。

  • 二、课程内容要点

    （1）Java程序基本结构。
    （2）Java类继承与多态。
    （3）Applet和Graphics。
    （4）图形界面(GUI)。
    （5）I/O流API。
    （6）线程、网络。
