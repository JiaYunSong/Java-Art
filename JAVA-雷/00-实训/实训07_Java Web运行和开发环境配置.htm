1. 一、目的

    学习如何搭建Java Web应用开发环境，以及如何使用集成开发工具开发Java Web应用。 

  2. 二、课程相关

    1.Java WEB 开发软件和硬件环境。

    2.下载和安装JDK。

    3.下载安装netbeans，下载安装eclipes。

  3. 三、演练

    • Java开发工具包（JDK）。

      下载（http://www.oracle.com/technetwork/java/javase/downloads/index.html）Java开发工具包（JDK）安装文件，如jdk-8u45-windows-i586.exe到计算机的某个磁盘如D盘，或者某个文件夹如e:\temp。然后，打开运行安装文件执行JDK安装。安装完成后JDK存放于C:\Program Files\目录下，例如下图所示。

    • 安装集成开发工具netbeans。

      下载（http://www.netbeans.org）netbeans安装文件，如netbeans-8.1-windows.exe到计算机的某个磁盘如D盘，或者某个文件夹如d:\temp，然后双击执行安装。开始安装时点击[定制]按钮打开选择项对话框，选择如图所示的选项，然后开始安装进程直到结束。

      安装完成后在桌面会看到NetBeans IDE 8.1图标，双击图标就可以启动NetBeans IDE 8.1。

    • 安装集成开发工eclipse。

      下载（http://www.eclipse.org）eclipse安装文件，如eclipse-jee-kepler-SR1-win32.zip到计算机的某个磁盘如D盘，或者某个文件夹如d:\temp，然后双击执行安装文件解压缩到E:磁盘，如图。

      解压缩完成后会在E:盘看到文件夹E:\eclipse，如图所示。

      双击[eclipse.exe]就可以启动eclipse。

    四、训练

  1. 1.启动NetBeans IDE 8.1，创建Web应用MyFirstWeb。

  2. 2.启动eclipse，创建Web应用MyFirstWeb。

  3. 3.启动NetBeans IDE 8.1，创建Web应用MySQLWebApp。下载mysql-connector-java-5.1.23-bin.jar，把mysql-connector-java-5.1.23-bin.jar添加到MySQLWebApp。
