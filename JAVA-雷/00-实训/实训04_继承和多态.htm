• 一、目的

    (1).进一步熟悉Java开发环境的使用。

    (2).通过实践练习初步了解Java程序的构成和编写方法。

    (3).学习通过继承构建Java类。

    (4).学习异接口、抽象类。

    (5).常用Java类（API）使用。

  • 二、课程相关

    （1）Java程序基本结构。

    （2）Java类继承与多态。

    （3）Applet和Graphics。

    （4）简单图形界面(GUI)。

    （5）I/O流API使用。
